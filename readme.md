# Emoji Images

Emoji images npm package. Returns map of emoji image files.

![Build](https://img.shields.io/travis/rumkin/emoji-img.svg)

## Installation

Install via npm:

```bash
npm i emoji-img
```

## Usage

Usage example:

```javascript
const emojis = require('emoji-img');

emojis.get('smiley'); // Full local path to smiley.png
```
